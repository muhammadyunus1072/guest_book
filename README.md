## Run Locally

Clone the project

```bash
  git clone https://github.com/muhammadyunus1072/guest_book.git
```

Go to the project directory guest_book

```bash
  cd guest_book
```

Install Dependencies

```bash
  composer install
```

```bash
  npm install
```

Start Vite

```bash
  npm run dev
```

Start Laravel

```bash
  php artisan serve
```
